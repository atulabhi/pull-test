PROJECT UPDATES

  • The OpenEBS community has been active at deploying and getting a feel of OpenEBS on different platforms like Amazon EC2 Cloud, Core OS, Rancher OS. A big thank you to our openebs-users: Maikotz and Eddy. We have made it to bench mark OpenEBS using percona server and client containers. Checkout our tutorial Running percona pod on OpenEBS. 
  • This month has been seeing a lot of new traction towards Container Native Storage in the community at large. Please refer to the discussion here on Reddit, and our take on this phenomenon in these blogs - Cloud Native Storage and Issues with Connected Storage Options.
  • Based on recent talk by William Kennedy at GopherCon India on Package Oriented Design for Go,we have refactored our code to create a Go Kit Project.
